## Mini example

`this library will help you out of this`

```
function() {
    return {
        tag: "div",
        props: {
            id: "key",
            class: "comp"
        },
        child: [
            {
                tag: "p",
                child: "This is ORVE"
            },
            "<p>object-reactive library</p>"
        ]
    }
}
```

`do this`

```
<div id="key" class="comp">
    <p>This is ORVE</p>
    <p>object-reactive library</p>
</div>
```
